# nchain.pos-grafana

## Custom urls

```
export CHAIN_ID=evmos_9000-1
export NODE_DOMAIN="domain1 domain2 domain3 domain4"

./update_urls.sh
```


## Local run

- Start

```
docker-compose up -d
```

- Stop

```
docker-compose down
```

## Default access

- Dashboard: http://localhost:3000
- User: admin/admin
